# 欢迎使用山东科技大学开源镜像站

山东科技大学开源软件镜像站是一个致力于普及开源软件，方便校内外用户高效访问开源项目、提升工作和科研效率的公益计划。由 SDUSTLUG运行维护。

# 网络指南

得益于学校信息化部门的支持，目前镜像站已支持校内外 IPv6/IPv4 的访问。

为保障您的服务质量，若您在其他高校，建议优先选择地理位置相邻、设立时间较长、内容更完善的教育网镜像站（可见教育联合镜像站列表 (opens new window)）。

  • 校内用户： mirrors.sdust.edu.cn (v4/v6自动选择) 为确保您能通过校园网连接，请使用DHCP获取到的DNS地址。
  • 校外用户： mirrors.sdust.edu.cn (v4/v6自动选择)
